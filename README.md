## Introducción  
Este repositorio contiene los apuntes, prácticas y ejercicios desarrollados a lo largo del curso de **Graficación**, cuyo objetivo principal es comprender los fundamentos del modelado, la representación visual y la interacción mediante herramientas computacionales.  

El contenido se encuentra organizado en carpetas que agrupan diferentes prácticas realizadas en clase. Estos ejemplos están implementados haciendo uso del lenguaje de programacion Python utilizando bibliotecas orientadas a gráficos, manipulación de imágenes y detección mediante visión por computadora.
